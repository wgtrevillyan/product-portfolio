# Image assets

Store site images in this folder using the following structure:

| Directory | Use for |
|-----------|---------|
| `./` | Shared assets (header images, favicon, og:image, etc.) |
| `companies/` | Company logos and hero images for company pages |
| `products/` | Product thumbnails, detail images, and gallery images |
| `patents/` | Patent-related images |

Reference in HTML with relative paths, e.g.:
- `images/header.png`
- `images/companies/acme-logo.png`
- `images/products/product-detail.webp`
