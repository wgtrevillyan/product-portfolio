# Image assets

Store site images in this folder using the following structure:

| Directory | Use for |
|-----------|---------|
| `./` | Shared assets (logo.svg, headshot.webp variants, home-header.png, favicon, etc.) |
| `companies/` | Company logos, companies-header.webp hero image |
| `products/` | Product thumbnails, products-header.webp hero image |
| `patents/` | Patent-related images, patents-header.webp hero image |

Reference in HTML with relative paths, e.g.:
- `images/home-header.png`
- `images/companies/companies-header.webp`
- `images/products/products-header.webp`
- `images/patents/patents-header.webp`
